# About

`index-ai-validator` is maintained by Jordach Makaya.

Jordach's work focuses on AI infrastructure for insurance claims workflows,
where agent-facing systems need to be reliable, inspectable, and testable.

`index-ai` exists as an experimental specification for exposing AI-readable
website metadata, Shadow Index graphs, and clean content endpoints. It is not a
formal standard.

`index-ai-validator` is the free CLI validator for that experimental spec. It
helps developers check whether a public website exposes the expected Level 1
and Level 2a layer without promising traffic, certification, SEO ranking, or
legal control over AI agents.
