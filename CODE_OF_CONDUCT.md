# Code of Conduct

This project should be a useful, respectful place to discuss `index-ai-validator`, AI-readable website infrastructure, documentation, and related tooling.

## Expected behavior

- Be respectful and constructive.
- Focus feedback on the work, not the person.
- Assume good intent, but accept correction when impact and intent differ.
- Keep technical disagreements specific and evidence-based.
- Respect privacy. Do not post private data, credentials, tokens, internal URLs, or personal information.

## Unacceptable behavior

- Harassment, insults, threats, or personal attacks.
- Discriminatory language or conduct.
- Publishing private information without permission.
- Sharing secrets, credentials, or exploit details in public issues or discussions.
- Repeatedly derailing issues or pull requests after maintainers ask to return to scope.

## Reporting

If you see behavior that violates this code, report it through GitHub's repository moderation or security reporting tools when available.

Do not use public issues to report vulnerabilities, leaked credentials, private data exposure, or security-sensitive behavior.

## Enforcement

Maintainers may remove comments, close issues, restrict participation, or take other reasonable action to keep the project safe and focused.

Enforcement decisions should be proportionate, documented when practical, and focused on protecting contributors and users.
